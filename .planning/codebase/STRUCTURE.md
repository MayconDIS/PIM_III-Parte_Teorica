## English Version

# Structure

### Directory Layout
- `index.html`: The main academic report document.
- `style.css`: The externalized stylesheet for the report.
- `*.png` / `*.pdf`: Assets, UML diagrams, mockups, and logos used within the academic report.
- `Old documents/`: Archive folder.

### Key Locations
- All "code" (C# snippets, DDL scripts) is currently embedded within `index.html` as documentation. No separate source code tree exists in this particular directory.
